Core Building Rules
본진 규칙
본거지 전체를 제어하는 핵심 건물의 역할 정의
레벨 상한 제어
본진 레벨 < 각 시설 레벨 불가. 모든 건물의 최대 레벨 = 본진 레벨. 업그레이드 병목 설계.
침공 전투 목표
본진 점령·파괴 시 본거지 기능 일시 제한. 침공전의 최종 목표이자 방어전의 핵심 거점.
지구 확장
본진 레벨업 또는 특정 조건 달성 시 지구 범위 확장 가능. 확장 상한·조건은 추후 정의.
FLOW
화면 전환 흐름
대륙 뷰에서 줌인하여 전술 뷰까지, 그리고 본거지·거점 화면으로 진입하는 전체 네비게이션 구조.
